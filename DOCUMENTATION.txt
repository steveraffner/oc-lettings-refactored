Documentation du projet :

La documentation est disponible dans le repository GitHub :
https://github.com/steveraffner/oc-lettings-refactored/tree/main/docs

Documentation générée avec Sphinx.

Pour consulter la documentation localement :
1. Cloner le repository
2. cd docs
3. make html
4. Ouvrir _build/html/index.html dans un navigateur

La documentation comprend :
- Guide d'installation
- Architecture du projet
- Guide de déploiement
- Documentation des modules (autodoc)
