Introduction
------------

OC Lettings est une application web développée avec Django 4.2 qui permet de :

- Gérer un catalogue de locations immobilières
- Gérer des profils utilisateurs avec leurs villes favorites
- Fournir une interface d'administration intuitive
- Assurer un déploiement continu via CI/CD

Caractéristiques principales
-----------------------------

* **Architecture modulaire** : Code organisé en applications Django réutilisables
* **Tests complets** : Couverture de code > 80%
* **Qualité du code** : Validation avec flake8, respect PEP 8
* **Déploiement automatisé** : Pipeline CI/CD avec GitHub Actions
* **Monitoring** : Intégration Sentry pour le suivi des erreurs
* **Containerisation** : Déploiement Docker prêt pour la production

Technologies utilisées
-----------------------

- Django 4.2.11
- Python 3.13
- PostgreSQL / SQLite
- Docker & Docker Compose
- GitHub Actions
- Sentry
- Read The Docs
